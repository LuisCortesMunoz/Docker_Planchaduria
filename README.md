# Backend Flask para Render

Este backend permite:

- guardar estado activo/inactivo
- guardar cantidad n
- subir fotos
- listar fotos
- guardar datos en Firebase Realtime Database

## Archivos necesarios
- app.py
- requirements.txt
- Dockerfile
- serviceAccountKey.json

## Ejecutar local
```bash
pip install -r requirements.txt
python app.py
